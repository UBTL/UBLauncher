# UBLauncher
Another Game Launcher for Linux (though there is also a Windows build too) focused towards Visual Novels / Galgames to make fellow weeb players life in Linux specially newcommers less miserable trying to get such archaic Nipponese Outdated Technology.

# Prerequisites:
You still need to pre-install WineTricks [https://github.com/winetricks/winetricks#installing](url)
and Umu-Run [https://github.com/Open-Wine-Components/umu-launcher#packaging](url) for your specific Linux distro.
If you are running some of those "Gaming Linux Distros" like CachyOS, Nobara or similar they tend to come pre-installed
already with WineTricks and Umu-run, just type on your console " umu-run " and/or " winetricks " and if you didn't got an
error it means they are preinstalled already.

Otherwise for Debian/Ubuntu based distros download the .deb file off [https://github.com/Open-Wine-Components/umu-launcher/releases](url)  and run in your command line where you download it:
sudo dpkg -i python3-umu-launcher_*.deb

And for Fedora based distros:
sudo dnf install python3-umu-launcher_*.rpm

# Optional:
If you want to monitor FPS, CPU, RAM, VRam and Temps Readings, you can install https://github.com/flightlessmango/MangoHud#installation---pre-packaged-binaries or look up your distro install of it on google so you can launch games with it, this is mainly for 3D Games stuff and such though, specially those made with stuff like Unreal Stutter 5 or similar.
